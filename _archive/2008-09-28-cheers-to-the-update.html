I proudly presents you with Comicalistic 0.5.5. Although there isn't too much of new features on this theme, it certainly has just gone through a few "major" fixes for cross browser usability.

Read more to find out more about the updates and the new comic strip for IE 6 users.

The following is the break down of the updates that I've made.

  • Fixed PNG transparency problem (by replacing the pngs with non-anti-aliazed gif transparent images using IE 6 conditional comments)
  • And layout problems for Internet Explorer 6 for some idiots who still use that. (that does not include you if you're viewing it using IE6 only because your idiotic company didn't update the browser)
  • Changed "mmhan" word of blog's logo to the logo that I intend to use for myself.
  • Updated the speech bubble.
  • Added conditional speech bubble especially for IE6
  • Fixed big ass textarea of Post comment section for Safari

And these few are some updates that I've made along the way since the launch,

  • Added subscribe to comments
  • Added socialable plugin for sharing (although I doubt anyone would use that)
  • Updated comments layout with Gravatars for the comment author
