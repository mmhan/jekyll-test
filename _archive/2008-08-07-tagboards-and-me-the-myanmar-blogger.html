ကၽြန္ေတာ္မ်ား Blog စေရးျဖစ္တုန္းကေပါ့ေလ.. Internet နဲ႕ပါတ္သက္တဲ့ knowledge လည္း သိပ္မရွိသလုိ blog ဆိုတာ ဂ်ိဳနဲ႕လားဆိုသလိုကို ျဖစ္ေနတာေပါ့ေလ။ ဒီေတာ့ နီးရာ စလံုး သူငယ္ခ်င္းေတြရဲ႕ blog ကိုယ္သြားၾကည့္ သူတုိ႕ေတြဘာလုပ္လဲ အတုယူခဲ့ရတာေပါ့။ အဲ့ဒီ့မွာ ဘာကိုသြားေတြ႕သလဲဆိုေတာ့ Tag Board။ အင္မတန္မိုက္တယ္ဗ်ာ။ ဟုိလူက ေရးသြားလုိက္ ဒီလူက ေရးသြားလုိက္နဲ႕။ ဘယ္ရမလဲ ျမတ္မင္းဟန္တုိ႕လဲ ေကာက္ထည့္လုိက္တာေပါ့ လာေလ့ေရာ့ Tagboard။

အဲ... ဒါေပမယ့္ အခုအခ်ိန္မွာ ျပန္စမ္းစစ္ၾကည့္ေတာ့ ေၾသာ္ ငါမွားခဲ့ပါလားေလလုိ႕။

ဘာေၾကာင့္မွားလဲဆိုရင္ေတာ့ သူတို႕ေတြ blog ေရးပံုနဲ႕ ကၽြန္ေတာ္တုိ႕ ျမန္မာ blogger ေတြ blog ေရးပံုကြာျခားခ်က္ေတြေၾကာင့္ပါပဲ

ကၽြန္ေတာ့္ သူငယ္ခ်င္း စလံုး blogger တုိင္းမွာ blog တစ္ခုေတာ့ရွိပါတယ္။ သူတုိ႕ေတြ English လုိေရးပါတယ္။ (စလံုးသံ ခပ္၀ဲ၀ဲေလးနဲ႕ေပါ့) သူတုိ႕ ေရးတာ သူတို႕ ဘာလုပ္တယ္ ဘယ္သြားတယ္ ဘယ္လာတယ္ စသျဖင့္ အလာပ သလာပ၊ တခါတေလ စိတ္ညစ္လုိ႕ေရးတဲ့ စာရႈပ္တစ္စု ႏွစ္စု၊ youtube မွာေတြ႕လုိ႕ ၾကိဳက္တဲ့ video clip တစ္ခုႏွစ္ခု၊ ဒါမွမဟုတ္ ဟိုတေလာကၾကည့္လုိ႕အရမ္းၾကိဳက္တဲ့ ရုပ္ရွင္၊ tv ဇာတ္လမ္းတြဲ စသျဖင့္ personal အင္မတန္က်တဲ့ theme ေလးေတြနဲ႕ေပါ့။

ကၽြန္ေတာ္တုိ႕မ်ားေတာ့ ေရးလုိက္ရင္ တျခားနာမည္ၾကီး blog ေတြလုိက္ဖတ္၊ သူတို႕ေရးတာေတြ သေဘာက်လို႕ ျပန္မွ်ေပးတာမ်ိဳးကတစ္ခု၊ တခါတေလ ၀တၳဳတုိတစ္ပုဒ္ႏွစ္ပုဒ္ကပါေသး၊ တခါတေလ ကဗ်ာတစ္ပုဒ္ႏွစ္ပုဒ္၊ တခါတေလ diary လုိလုိ ဘာလုိလုိ စာစုေလး တစ္စုႏွစ္စု ေျဗာက္ေသာက္ေတာ့ ႏုိင္လြန္းတာေပါ့။

သူတို႕မ်ား စလံုးေတြရဲ႕ blog မွာေတာ့ ကိုယ့္အတန္းထဲက သူငယ္ခ်င္း တကယ္သိတဲ့သူေတြ ဖတ္ဖုိ႕ေရးတဲ့ blog ျဖစ္တဲ့အတြက္ အသိေတြကလာဖတ္ၾက ေနာက္ေန႕ဘယ္သြားမယ္၊ ဘယ္သူေတာ့ဘာျဖစ္သြားျပီ စသျဖင့္တသြယ္ စိတ္ညစ္ေနတယ္ဆို ႏွစ္သိမ့္သြားၾက အခက္အခဲရွိတယ္ဆို အၾကံဥာဏ္ေလးေတြေပးသြားၾက ပစၥည္းသစ္တစ္ခုခု၀ယ္ထားတာကို ထုတ္ၾကြားထားတယ္ဆို ေနာက္ရင္ေနာက္ မေနာက္ရင္ မိုက္တယ္ကြာ ငါလည္း ဒီလုိ၀ယ္ရင္ေကာင္းမလားစဥ္းစားေနတာ စသျဖင့္တဖုံ ေျပာျပီးေတာ့ ျပန္သြားၾကတာေပါ့။

ကၽြန္ေတာ့္မွာေတာ့ သူငယ္ခ်င္းေတြထဲမွာ blog မရွိတဲ့ သူကခပ္မ်ားမ်ား၊ ရွိတဲ့သူမ်ားၾကေတာ့လည္း တစ္အိမ္ထဲမွာ ေနခဲ့ၾကတဲ့ ေကာင္ေတြ၊ အဲ့ေတာ့ ဘယ္သူကမ်ား စကား ေထြေထြထူးထူး ေျပာၾကမလဲ။ မသိတဲ့ တခ်ိဳ႕မ်ားၾကေတာ့ ေျပာစရာ ခပ္မ်ားမ်ား စားစားကလည္း မရွိ ကိုယ္နဲ႕သိတဲ့လူလည္း မဟုတ္ေတာ့ ဟိုလုိလုပ္ပါလား ဒီလုိလုပ္ပါလားလည္း မေျပာသင့္ဆိုေတာ့ ေရာက္လာတယ္ ဖတ္သြားတယ္လုိ႕ ေရးသြားၾက တခ်ိဳ႕မ်ားၾကေတာ့ ဖတ္လုိ႕ေကာင္းတယ္ ဘာရယ္ေရးၾကတာေပါ့။ အဲတခ်ိဳ႕မ်ားၾကေတာ ဒီ Link ေလးကို လာပါဦး၊ လာဖတ္သြားတယ္ ျပန္လာဖတ္ဦး ဆိုတာမ်ိဳးေပါ့။ ေနာင္မ်ားၾကေတာ့ ဟိုလူၾကီးေတြ ဘာျဖစ္လုိ႕ ဘာျဖစ္တယ္ဆိုတာနဲ႕ ဒီရက္ဟာ ဘယ္လုိ ရက္ေကာင္းရက္ျမတ္ျဖစ္တဲ့ အတြက္ ဘယ္သူ႕ကို ဂါရ၀ျပဳၾကပါ ဘယ္သူ႕ကို ရႈံ႕ခ်ပါ စသျဖင့္ ျပန္ၾကည့္မိတဲ့အခါၾကေတာ့ ကၽြန္ေတာ့္ Tag board ဟာ သူငယ္ခ်င္းမ်ားေရးတဲ့ တကယ့္ genuine ျဖစ္တဲ့ dialogue ေတြအစား ဘတ္စ္ကားဂိတ္က ေၾကာ္ျငာကပ္တဲ့ board လုိျဖစ္ေနေတာ့ပါတယ္။ ဒါ့အျပင္ tagboard ေတြရဲ႕ server မ်ားကမေကာင္းေတာ့ အဲ့ဒါေလးတစ္ခု တပ္လာဖုိ႕အတြက္နဲ႕ page load time က သိသိသာသာကို ၾကာလာပါတယ္။

ဒီေတာ့ ကၽြန္ေတာ့္ရဲ႕ ဒီ blog မွာ Tag board မထားေတာ့ပါဘူးလုိ႕။ တကယ္လုိ႕ post ကိုၾကိဳက္ရင္ post နဲ႕ ပါတ္သက္တဲ့ အေၾကာင္းမ်ားေျပာခ်င္ရင္ comment ထားခဲ့ႏုိင္ျပီး တျခားတျခားေသာ အေၾကာင္းမ်ားေျပာခ်င္ရင္ email ပို႕ႏုိင္ေအာင္ contact page ေလးတစ္ခု ထည့္ပါမည္ ထည့္ျပီးျပီ ျဖစ္ေၾကာင္း သူငယ္ခ်င္းမ်ား ကၽြန္ေတာ့္ကို အလာပ သလာပေျပာခ်င္ပါက online 24x7 နီးပါ:ရွိသျဖင့္ အနီေရာင္ေလးေတြထဲမွာ ရွာၾကည့္ႏုိင္ေၾကာင္း...

အဲ့လုိ Tagboard ကို spam မလုပ္သင့္ေၾကာင္းကိုလည္း လုပ္တတ္သူမ်ားကို အမ်ားျပည္သူကိုယ္စား ေတာင္းပန္ရင္း...
