ဒီလုိပါ ျပည္ပေရာက္ေနက်ေတာ့ ျမန္မာဟင္းေလး ဘာေလး စားခ်င္က်တာေပါ့... ျမန္မာဟင္းခ်က္နည္း website ေလးဘာေလး ရွိလားလုိ႕ပါဗ်ာ... မဟုတ္ရင္လဲ အဲ့လုိ site ေလးတစ္ခုလုပ္တင္ရင္ ေကာင္းမလား ဘယ္လုိေနလဲ စဥ္းစားမိလုိ႕ပါ... ရွိျပီးသားဆိုလဲ ေျပာပါဦး info ေလးေတြ။ ကိုယ္က ေကာင္းေကာင္း မခ်က္တက္ဆိုေတာ့ haha

ကၽြန္ေတာ္ကုိယ္တုိင္ကေတာ့ ဟင္းခ်က္သူတစ္ေယာက္ မဟုတ္တဲ့ အတြက္ စံုစံုလင္လင္ information ေတာ့မရွိပါဘူး။ ဒါေပမယ့္ ကၽြန္ေတာ္ ဟိုအရင္တုန္းက ေတြ႕မိ မွတ္မိ bookmark လုပ္မိတဲ့ ျမန္မာ ထမင္းဟင္းနဲ႕ သက္ဆုိင္တဲ့ blog ေလးတစ္ခုနဲ႕ website ေလးတစ္ခုကို ေ၀မွ်လုိက္ပါတယ္။

ေမတၱာရပ္၀န္း မီးဖုိေဆာင္ [ LuLuCooking.blogspot.com ]

ဒီ site ေလးကေတာ့ လုလုဆိုတဲ့ Belgium မွ အစ္မတစ္ေယာက္က ဦးေဆာင္ကာ ရွင္မင္း၊ ယြန္းမင္းငယ္၊ တုတ္ၾကီး စသူတုိ႕က ျမန္မာဟင္း ခ်က္နည္း ျပဳတ္နည္းမ်ားကို အပတ္စဥ္ တစ္ပါတ္ကို ႏွစ္ခုေလာက္စီ ဓါတ္ပံုနဲ႕တကြ တင္တဲ့ blog ေလးတစ္ခုပါ။ ေသေသခ်ာခ်ာ ရွင္းျပရုံမက ဓါတ္ပံုေလးေတြၾကည့္ရုံနဲ႕ ခ်က္ခ်င္းထလုပ္ခ်င္စရာေလးပါ။

သက္သက္လြတ္ဟင္းေတြ၊ အကင္ေမႊးေမႊးေလးေတြ၊ အသီးအရြက္သုပ္၊ အသားသုပ္ေတြ၊ အခ်ိဳပြဲေတြ၊ အသီးအရြက္၊ အသားငါး ခ်က္မလား ေၾကာ္မလား ေလွာ္မလား... အို လုပ္ခ်င္ရာလုပ္ အဲဒီ့မွာ recipes ေတြရွိတယ္ဗ်ာ။

Hsa*Ba - စားပါ [ Hsaba.com ]

ဒီ site ကေတာ့ England မွ မတင္ခ်ိဳေခ်ာရဲ႕ English ဘာသာႏွင့္ ျဖန္႕ေ၀ေရာင္းခ်မယ့္ ျမန္မာ့ အစားအစာ စာအုပ္ စားပါ ေခၚ Hsa*ba ဆိုတဲ့ စာအုပ္ေလးရဲ႕ website ေလးတစ္ခုပဲျဖစ္ပါတယ္။ စာအုပ္ကုိ promotion လုပ္ဖုိ႕ အတြက္ တင္ထားတဲ့ website တစ္ခု ျဖစ္ေသာ္ျငားလည္း recipes ဆယ့္ႏွစ္ခုေလာက္ကို ရက္ရက္ေရာေရာ တင္ထားေပးတာေတြ႕ရပါတယ္။

ဒါ့အျပင္ Interactive Solution company မွာ web developer တစ္ေယာက္ရဲ႕ skill နဲ႕ အညီ websiteကို flash ဒီဇုိင္းလွလွေတြနဲ႕ တစ္သြယ္၊ accessible ျဖစ္မယ့္ html page ေတြနဲ႕က တစ္မ်ိဳး လွလွပပ တင္ဆက္ျပထားတာေတြ႕ရမွာပါ။ (Now I'm almost embarrassed that my site is so crappy compared to that.)

ဒါနဲ႕... ထူးထူးဆန္းဆန္း recipe တစ္ခုေတာင္ေတြ႕ရေသးတယ္.. ဘာတဲ့... ေထာပတ္သီးေရခဲမုန္႕ဆိုပဲ.. ကဲေျပာေနၾကာပါတယ္ Hsaba.com ကိုသာသြားျပီးေတာ့ အရသာရွိတဲ့ recipes ေတြလဲၾကည့္ရင္း တစ္ခါတည္း စာအုပ္ထြက္တဲ့အခ်ိန္ သိဖုိ႕ mailing list ကိုသာ join ထားလုိက္ပါေတာ့။

ေျပာရင္းနဲ႕ေတာင္ အဲ့ဒီ့ site ေတြၾကည့္ရင္း ဗိုက္ဆာလာျပန္ျပီ... ကၽြန္ေတာ္လည္း အားက်မခံ ေခါက္ဆြဲေလးျပဳတ္စားလုိက္ဦးမယ္။ ခ်က္လုိ႕ ျပဳတ္လုိ႕ ျပီးသြားလုိ႕ ေက်းဇူးတင္ခ်င္တယ္ဆိုရင္ေတာ့ ထမင္းေခၚေကၽြးလုိ႕ရပါတယ္။ အားမနာတမ္း ပါးျဖဲနားျဖဲ စားေပးပါ့မယ္။

ေၾသာ္... ဒါနဲ႕ တျခား resource ေကာင္းေကာင္းေလးမ်ားရွိရင္လည္း comment ေလးမ်ားေပးလုိ႕ အားလံုးကို မွ်ေ၀သြားလုိက္ပါဦးလုိ႕..

ျပီးေတာ့ တျခားတျခားေသာ အေၾကာင္းအရာမ်ား သိခ်င္ရင္ contact form မွတဆင့္ ေတာင္းဆိုပါက ကၽြန္ေတာ္ သိမိ မွတ္မိသေလာက္ တင္ေပးသြားပါမယ္ဆိုတဲ့ အေၾကာင္း ကတိေပးရင္း...
