စလံုးမွာ တစ္ေနရာကေန တစ္ေနရာသြားဖုိ႕ MRT နဲ႕ အဆင္မေျပျဖစ္ေနသလား၊ လူေတြျပည့္ၾကပ္ေနတဲ့ MRT ကို ေန႕တုိင္းစီးေနရတာ ျငီးေငြ႕လာျပီလား၊ Bus စီးခ်င္ေပးမယ့္ ဘယ္ကားစီးရမွန္း မသိ ျဖစ္ေနေသးသလား၊ ဘယ္ကားစီးရမယ္လုိ႕ ျပေပးတဲ့ StreetDirectory.com က LTA နဲ႕ ျပႆနာတက္ျပီး ဟိုတစ္ရက္က service ရပ္သြားျပီးတဲ့ေနာက္ ေၾကျငာ ပရပြနဲ႕ျပန္တတ္လာတဲ့ အတြက္ စိတ္ပ်က္ေနလား။

ဒါဆို GoThere.sg ကို ေျပာင္းသံုးၾကတာေပါ့...

GoThere.sg ရဲ႕ Description ကေတာ့ ဒီလုိဗ် -

တစ္ေနရာရာကို သြားခ်င္သလား? ဘယ္ေနရာလဲသာေျပာ ဘယ္လုိသြားရမယ္ဆိုတာ ေျပာျပမယ္။ ေနရာတစ္ခုခုကို ရွာေနသလား၊ ဘာလဲသာေျပာ ဘယ္မွာလဲေျပာျပမယ္

Site က AJAX driven ျဖစ္တဲ့အျပင္၊ Google Map ကို အေျခခံျပီး ေရးထားတဲ့အတြက္၊ StreetDirectory.com ထက္အရမ္းျမန္ျပီး သံုးရတာလဲ လြယ္ပါတယ္။

ဒါ့အျပင္ ကိုယ္စသြားခ်င္တဲ့ ေနရာ (A) ကေန (B) ကိုသြားဖုိ႕ result ကို Bus နဲ႕ သြားမလား၊ MRT နဲ႕သြားမလား ေရြးလုိ႕ရျပီး အခ်ိန္ကိုလည္း တြက္ေပးတယ္ ဘယ္ေလာက္က်မယ္ကိုလဲ တြက္ေပးတယ္ေကာ။

မေက်နပ္ေသးဘူးလား၊ ထြက္လာတဲ့ result ေျမပံုကေန A ေလးပဲျဖစ္ျဖစ္ B ေလးပဲျဖစ္ျဖစ္ Drag And Drop လုပ္လုိက္တာနဲ႕ result အသစ္ျပန္ရွာေပးေသးတယ္။

တစ္ခုေတာ့ရွိတယ္... Drop လုပ္တဲ့အခါ ေရထဲပစ္ခ်တာတုိ႕၊ Malaysia ဘက္ထဲ ခ်လုိက္တာမ်ိဳးေတာ့ မလုပ္ေၾကးေပါ့။

ကုိယ္က လူမသိ သူမသိ မသြားတတ္ မလာတတ္ ေနရာတစ္ခုမွာ event တစ္ခု လုပ္ခ်င္သလား၊ ကုိယ္လုပ္ခ်င္တဲ့ ေနရာေလးကို အဲ့ဒီ့ site မွာေရြး။ ျပီးရင္ "Embed this map on your website" ဆိုတာေလးကိုႏွိပ္လုိက္၊ (dimension ေတြဘာေတြ customize လုပ္လုိက္ဦး) ျပီးတာနဲ႕ သူေပးတဲ့ code ေလးကုိယူ ကိုယ့္ Event အေၾကာင္းေရးထားတဲ့ page ေလးမွာ embed လုပ္ေပးလုိက္။

ဒါဆိုရင္ ကိုယ့္ event ကို လာမယ့္သူေတြ အကုန္ မလာတတ္ရင္ အဲ့ဒီ့မွာ တစ္ခါတည္း ရွာလုိက္လုိ႕ရသြားျပီဗ်ာ...

ဒါပဲလားဆိုေတာ့ မဟုတ္ေသးဘူးဗ်ာ၊ သြားရင္းလာရင္း ဘယ္လုိသြားရမလဲ သိခ်င္ေသးလား mobile version ကိုလဲ GoThere.sg/m မွာ သံုးလုိ႕ရေသးတယ္ ဆိုပဲ။

ေၾသာ္... ေျပာရင္းေျပာရင္းနဲ႕ လူက ျမန္မာႏုိင္ငံ မီးရထားေပၚက လ်က္ဆားေရာင္းတဲ့လူ အသံထြက္လာျပီ။ အရင္တုန္းက အလုပ္အကိုင္ေတြ လူမသိခင္ ေတာ္ရုံနဲ႕ရပ္မွ။ ကိုယ့္ကို ဒီလုိေျပာလုိ႕ သူတုိ႕ကလည္း ဘာမွေပးမွာမဟုတ္ေတာ့ အခက္သား။

ညဘက္အျပင္ထြက္တတ္ေတာ့ NR လုိင္းေတြ အတြက္ပါ ျပေပးရင္ အေကာင္းသားလုိ႕ စဥ္းစားမိတယ္။ Feedback ေတာ့ေပးလုိက္ျပီ။ ေနာက္ပိုင္း NR လုိင္းေတြ ေတြ႕လုိ႕ကေတာ့ ကၽြန္ေတာ္ အတုိ႕အေထာင္လုပ္လုိ႕သာ မွတ္ေပေတာ့ ကရုိ႕။
