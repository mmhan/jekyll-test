Limewire တုိ႕ Aries တုိ႕နဲ႕ download လုပ္ေနၾကပဲ ဆိုျပီး ဆက္မဖတ္ပဲ ေနလုိ႕ကေတာ့ မွားမယ္ေနာ္...

ဟိုအရင္ တစ္ေခါက္က ေျပာသလုိပဲ အဲ့ဒါေတြ က virus ေတြ၊ spyware၊ malware ေတြရတတ္တယ္ေလ။ ျပီးေတာ့ အခံ့မသင့္လုိ႕ ဖမ္းၾကျပီဆိုရင္ ဒါမ်ိဴးေတြက အရင္ဆံုး စဖမ္းၾကတာမ်ိဴး ဆိုေတာ့ အတတ္ႏုိင္ဆံုး အဲ့ဒါမ်ိဳး မသံုးတာ ေကာင္းမွာေပါ့။

ကၽြန္ေတာ္ေျပာမယ့္ ဒီ Favtape ကေတာ့ အဲ့လုိ ဘာ p2p software ေတြမွ သြင္းစရာမလုိပဲ၊ ဒီအတုိင္း Online မွာ ေနာက္ဆံုးေပၚ နာမည္ၾကီးသီခ်င္းေတြ၊ ေခတ္ေဟာင္း သီခ်င္းေတြကို နားေထာင္လုိ႕ရတဲ့ website ေလးတစ္ခုေပါ့။ အဖြဲ႕ တစ္ဖြဲ႕ ႏွစ္ဖြဲ႕၊ အဆိုေတာ္ တစ္ေယာက္ ႏွစ္ေယာက္တည္း သည္းၾကီးမည္းၾကီး ၾကိဳက္တာမ်ိဳးမဟုတ္ပဲ သီခ်င္းေကာင္းမွန္သမွ် နားေထာင္ခ်င္တဲ့ သူမ်ိဳးအတြက္ေတာ့ အေကာင္းဆံုးေပါ့။

(အဆံုးထိဆက္ဖတ္ေနာ္... ဘယ္လုိ ေဒါင္းလုဒ္လုပ္ရလဲ ဆုိတာပါ တခါတည္း တုိးတုိးတိတ္တိတ္ ေျပာျပမယ္) :D

Favtape ကို ဘယ္လုိသံုးမလဲ

လြယ္ပါတယ္၊ Favtape.com ကိုသြားလုိက္ သူ႕မွာ အေကာင္းတကာ့ အေကာင္းဆံုးဆိုတဲ့ သီခ်င္းေတြကို list လုပ္ေပးထားလိမ့္မယ္။ iTunes Music Store မွာ အေရာင္းရဆံုး သီခ်င္း ေပါင္းစံု (ဒါမွမဟုတ္) Hip-Hop လား၊ Alternative လား ၾကိဳက္တာလာ အပုဒ္ ၁၀၀ စီ list လုပ္ေပးထားတယ္။

ဒါတင္ပဲလားဆိုေတာ့ မဟုတ္ေသးဘူး၊ ႏွစ္အလုိက္ အေကာင္းဆံုး သီခ်င္းေတြ၊ ႏွစ္အလုိက္ အေကာင္းဆံုး artist ေတြရဲ႕ သီခ်င္းေတြကိုလဲ click ႏွစ္ခ်က္ေလာက္နဲ႕ အစအဆံုး နားေထာင္လုိ႕ရတယ္။

မေက်နပ္ေသးဘူးလား၊ Last.fm တုိ႕ US က resident ေတြကလြဲျပီး နားေထာင္ခြင့္ မေပးေတာ့တဲ့ Pandora တုိ႕ သံုးတယ္ဆိုရင္ အဲ့ဒီ့မွာ Love လုပ္ထားတဲ့ သီခ်င္းေတြ၊ Bookmark လုပ္ထားတဲ့ သီခ်င္းေတြကိုလည္း Favtape က import လုပ္ျပီး အစအဆံုး နားေထာင္လုိ႕ရတယ္။

ေရာသမေမႊ Interface

အဆန္းထြင္ျပီး ေျပာတယ္မထင္နဲ႕ဗ်ာ။ ေအာက္က Interface ပံုေလးကို ၾကည့္ၾကည့္လုိက္ပါဦး။

Get MP3, CD, iTunes, Ringtones, Ebay link ေတြနဲ႕ အဲ့ဒီ့ သီခ်င္းရဲ႕ ပါတ္သက္ရာ ပါတ္သက္ေၾကာင္းေတြ ၀ယ္လို႕ရတယ္။ (ကၽြန္ေတာ့္လုိ ငမြဲေတြကိုေတာ့ အားမေပးဘူးဗ်ာ၊ သီခ်င္းေတြက အမ်ားၾကီးရယ္၊ ကုန္ေပါက္ပဲ)

တျခား Link ေတြနဲ႕ သူတုိ႕ရဲ႕ Function ေတြက -

Lyrics - Lyricswiki.org က သီခ်င္းရဲ႕ Lyrics ရဲ႕ Link

Video - Youtube က သီခ်င္းရဲ႕ MTV ရဲ႕ Link

Artist - သီခ်င္း အဆိုေတာ္ရဲ႕ Favtape မွာရွိသမွ် သီခ်င္းေတြ

Similar - သီခ်င္းနဲ႕ ခပ္ဆင္ဆင္တူတဲ့ သီခ်င္းေတြ

Link, Email, Share နဲ႕ Twitter ကေတာ့ ရွင္းျပဖုိ႕ လုိမယ္ မထင္ပါဘူး။

ေဘးနားက Thumbnail ႏွစ္ခု - ညာဘက္က YouTube က MTV, ဘယ္ဘက္က Album Art က Amazon က အေခြ၀ယ္တဲ့ page.

ကုိယ္တုိင္ List လုပ္ဦးမလား

နားေထာင္ရုံနဲ႕တင္ အားမရတဲ့သူေတြအတြက္လည္း ကိုယ္ကိုယ္တုိင္ playlist တစ္ခု ျပန္လုပ္လုိ႕ရတယ္။

ငယ္ငယ္တုန္းကလုိေပါ့ ကိုယ္ၾကိဳက္တဲ့ သီခ်င္းေလးေတြကို ကက္ဆက္ေခြေလး CD ေခြေလး တစ္ေခြမွာ စုျပီးထည့္ၾကျပီးရင္ သူငယ္ခ်င္းလုိလုိ ဘာလုိလုိေလးေတြကို ေပးလုိ႕ ရတာေပါ့။ Mixtape လုိ႕ေခၚခ်င္ေပမယ့္ RIAAက ကြိဳင္ရွာခံလုိက္ရတဲ့ Muxtape ကလည္း ရွိေနျပန္၊ Mixtape.com ကလည္း သူမ်ားက park လုပ္ထားတယ္ဆုိေတာ့က အဆင္မေျပတာနဲ႕ Favtape ျဖစ္သြားပံုရပါတယ္။

လုိရင္းကို ျပန္ေျပာရရင္ေတာ့ playlist တစ္ခုလုပ္ဖုိ႕ အရင္ဆံုး Sign-up လုပ္ရမွာေပါ့ေလ။ ျပီးရင္ Favtape ေလးတစ္ခု create လုပ္၊ နားေထာင္ရင္း ရွာရင္းေဖြရင္းနဲ႕ ကုိယ့္ tape ထဲကို Add to tape ေလးႏွိပ္ျပီး ထည့္လုိက္ရုံပါပဲ။

ဘယ္လုိ အလုပ္လုပ္သလဲ

ဟုိရက္က Skreemr, Seeqpod, BeeMp3 နဲ႕ SongZa တုိ႕လုိ Internet ေပၚက ေထာင္ေပါင္းမ်ားစြာေသာ server ေတြမွာ လူေတြ တင္တင္ထားတဲ့ Mp3 file ေတြ index လုပ္ျပီး လူေတြကို search လုပ္လုိ႕ရေအာင္ လုပ္ေပးတဲ့ Search Engine ေတြ ေပၚလာခဲ့တယ္ေလ။ တခ်ိဳ႕တခ်ိဳ႕ေတြက Download ေပးျပီး တခ်ိဳ႕ေတြကေတာ့ Download မေပးပါဘူး။

အဲ့ဒီ့ အထဲက ေတာ္ေတာ္စံုစံုကို index လုပ္ရုံမက youtube video ေတြကေန အသံကိုပဲ extract လုပ္ျပီး playback လုပ္လုိ႕ရေအာင္လုပ္ေပးတဲ့ Seeqpod ရဲ႕ database ကို ပရုတ္ဆီ မည္သို႕လုပ္လုိက္သည္ မသိ ဆြဲစိျပီး ဖြင့္သမွ် သီခ်င္းေတြကို အဲ့ဒီ့ကေနတဆင့္ အရပ္ ရွစ္မ်က္ႏွာကေန လွမ္းဆြဲေပးျပီး ဖြင့္ေပးတယ္ ဆိုပါေတာ့။

တျခား Search Engine Site ေတြနဲ႕ မတူတာက သီခ်င္း အတူ ႏွစ္ပုဒ္သံုးပုဒ္ မထပ္ေအာင္လည္း  လုပ္ထားေပးတယ္။ (User ရဲ႕ Feedback မပါဘဲနဲ႕ အဲ့လုိ database ကို ဘယ္လုိ optimize လုပ္လဲ သိသူ တတ္သူမ်ားရွိရင္ သိခ်င္လုိ႕ ရွင္းျပ ေပးၾကပါဦးဗ်ာ။)

ဘယ္လုိ ေဒါင္းလုဒ္လုပ္မလဲ

ဒီအထိေရာက္ေအာင္ သည္းခံျပီး ဖတ္လာတဲ့ စာဖတ္သူကို နိဒါန္းမပ်ိဴးေတာ့ပဲ ေျပာျပမယ္ဗ်ာ။

Step #1 - Firefox ရဲ႕ Tools > DownThemAll! ဒါမွမဟုတ္ Tools > Downloads (Ctrl + J) window ေလးကို ဖြင့္ထား။

Step #2 - သီခ်င္းဖြင့္တဲ့ page က ကိုယ္ၾကိဳက္တဲ့ သီခ်င္းရဲ႕ ဧရာမ Title ၾကီးကို Drag လုပ္ျပီး ဖြင့္ထားတဲ့ window ေပၚကုိ Drop လုပ္လုိက္။ အဲ့ဒီ့မွာ Download စျပီ။

Step #3 (for DownThemAll!) - Download လုပ္လုိ႕ျပီးသြားရင္ fetch ဆိုျပီး ထြက္လာတဲ့ file ေလးကို .mp3 extension ပါတ့ဲ နာမည္ေလးေျပာင္းျပီး သိမ္းလုိက္ရုံပါပဲ။

Javascript ကို အဓိကသံုးျပီး run တဲ့ Favtape ကေတာ့ ေတြ႕မိသမွ် web2.0 service ေတြရဲ႕ Mash-up ေတြထဲမွာ ေတာ္ေတာ္ေလးကို impressive ျဖစ္တဲ့ mash-up တစ္ခုလုိ႕ေတာင္ ေျပာလုိ႕ရပါတယ္။

ကဲ... Favtape.com ကို စမ္းသံုးၾကည့္လုိက္ပါဦး...

သီခ်င္းေကာင္းေလးေတြ ေတြ႕ရင္လည္း Share လုိက္ပါဦးဗ်ာ။ Falling for you လုိ႕ နာမည္ေပးထားတဲ့ playlist ေလးကေတာ့ ကၽြန္ေတာ္ ဒီရက္ပိုင္း နားေထာင္ျဖစ္တဲ့ သီခ်င္းေလးေတြေပါ့ဗ်ာ။
