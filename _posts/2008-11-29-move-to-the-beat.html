I'm not sure if you guys have heard about my project, Move to the Beat, a game based on the animated movie, Sing to the Dawn.

Sing to the Dawn is a movie by Infinite Frameworks, backed by Media Development Authority Singapore. This movie is based on award winning novel and produced "locally" here in Singapore.

This project was done as my final year project in one semester time, creating rhythm based game using flash. I was responsible for programming of the game engine and compiling all the assets given to me. Even though this project was already finished since last year's December, its release has been delayed to accompany the release of the movie.

Oh well, enough talking. Click the link below to take a jump to the game page on Sing to the Dawn movie's site and have fun.
