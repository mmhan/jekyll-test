Google Reader မွာ Zawgyi-one ကို ျမင္ရဖုိ႕ အလြယ္ဆံုးနည္းကေတာ့ Stylish ဆုိတဲ့ Firefox plug-in ကိုသံုးျပီး tweak လုပ္ရမွာ ျဖစ္ပါတယ္။

Stylish ကုိ Download လုပ္ျပီး Firefox ကို restart ျပန္ခ်လုိက္ပါ။

[caption id="attachment_83" align="aligncenter" width="317" caption="Stylish Screen Shot"][/caption]

ျပီးရင္ေတာ့ Status bar ထဲမွာ Stylish Icon ေလးကို လုိက္ရွာျပီး click ႏွိပ္လုိက္ရင္ ေဘးမွာျပထားတဲ့ အတုိင္း menu ေပၚလာပါလိမ့္မယ္။ Menu ထဲက Write Style > Blank Style ကို ေရြးျပီး ေတာ့ ဒီ file ထဲက code ကို ကူးျပီးထည့္လုိက္ပါ။

အဲ့ဒီ့ Code က Google Reader ထဲမွာ ရွိတဲ့ Entry ေတြနဲ႕ Feed ေတြရဲ႕ Name ကို Zawgyi-one နဲ႕ျပေပးေအာင္ လုပ္တဲ့ code ပါ။ အဆင္ေျပသြားပါလိမ့္မယ္။ တကယ္လုိ႕ အဆင္မေျပဘူးဆုိရင္ Comment ထားခဲ့လုိက္ပါ။
