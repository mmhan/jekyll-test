စိတ္ကူးေလးေပါက္လုိ႕ ေရးခ်င္စိတ္ေလးရွိတုန္း ဂ်ိဳနဲ႕လား series အတြက္ အသစ္တစ္ခု ထပ္တင္လုိက္ပါတယ္။ ဒီ post မွာေတာ့ Web Browser ေတြ အေၾကာင္းကို ကြန္ပ်ဴတာ စသံုးတဲ့ user တစ္ေယာက္ အဆင့္ကေန၊ ပံုမွန္ အိမ္သံုး user ေတြ အထိ နားလည္ေအာင္ ရွင္းျပဖုိ႕ၾကိဳးစားရင္းနဲ႕၊ Web Browser ေတြ အၾကမ္းဖ်င္းအားျဖင့္ ဘယ္လုိ အလုပ္လုပ္တယ္၊ ဘာကို သံုးသင့္တယ္ ဆိုတာေလးပါ ေဆြးေႏြးသြားမွာပါ။

Web Browser ဆိုတဲ့ စကားလံုးကုိ မသိတဲ့သူ မရွိေလာက္ဘူးလုိ႕ ယံုပါတယ္... အကယ္၍ တကယ္မသိဘူး ဆိုရင္ေတာ့ တစ္ေယာက္ေယာက္ သြားေမးေပါ့... သူကေျပာပါလိမ့္မယ္... "ဒါေလးေတာင္ မသိဘူးလား.. မင္း Desktop ေပၚက e ဆိုတဲ့ icon ေလးနဲ႕ internet explorer လုိ႕ေရးထားတာကို web browser လုိ႕ေခၚတယ္"လုိ႕ ေျပာပါလိမ့္မယ္။

Internet explorer လုိ႕ ဘာလုိ႕ေခၚေနေသးလဲ၊ web browser လုိ႕ေခၚလုိက္လုိ႕ရတာပဲ

မရပါဘူး။ လြယ္ေအာင္ ေျပာရရင္ေတာ့ Blogger အမ်ားၾကီးရွိတဲ့ အခန္းထဲက  တစ္ေယာက္ကို ေဟ့ ကိုblogger လုိ႕ လွမ္းမေခၚပဲနဲ႕ ကိုဘယ္သူ မဘယ္၀ါ စသျဖင့္ ေခၚသလုိ၊ web browser ေတြအမ်ားၾကီးရွိတဲ့ အခ်ိန္မွာ Internet Explorer ကိုမွ web browser လုိ႕ ေခၚလုိ႕ မရပါဘူး။

ဘယ္လုိ? web browser ေတြ အမ်ားၾကီးရွိတယ္?

ဟုတ္ပါတယ္။ အသံုးအမ်ားဆံုးျဖစ္တဲ့ Windows မွာ နဂိုကတည္းက ပါျပီးသားျဖစ္တဲ့ Microsoft ရဲ႕ Internet Explorer, Mozilla ရဲ႕ Firefox, Opera, Mac ေတြမွာ ပါျပီးသားျဖစ္တဲ့ Apple ရဲ႕ Safari အျပင္ တျခားတျခား browser ေတြ အမ်ားၾကီးရွိပါတယ္။

Browser ေတြ ဘယ္လုိအလုပ္လုပ္ပါသလဲ?

Browser ေတြဟာ HTML ဆိုတဲ့ text file ေတြကို ဖတ္ျပီးေတာ့ <title> ဆိုတဲ့ tag မ်ိဴးေတြ႕ရင္ Window ရဲ႕ title အျဖစ္ျပေပးတယ္၊ <b> ဒါမွမဟုတ္ <strong> ကိုေတြ႕ရင္ Bold လုပ္ေပးတယ္။ Text file ေတြမွာ <a> ဆိုတာေလးေတြေတြ႕ရင္ သူ႕ရဲ႕ link ကို ျပေပးတယ္ ႏွိပ္လုိက္ရင္ အဲ့ဒီ့ link လုပ္ထားတဲ့ page ကို ဖြင့္ေပးတယ္၊ <img> ကိုေတြ႕ရင္ သူ႕ သက္ဆိုင္ရာ image ကိုျပေပးတယ္ စသျဖင့္ေပါ့။ HTML အျပင္ CSS ဆိုတဲ့ website ေတြကို ဘယ္လုိျပရမယ္ဆိုတဲ့ rule ေလးေတြကို website ေရးတဲ့သူက ထည့္ေပးရင္ အဲ့ဒီ့ browser တစ္ခုရဲ႕ ပံုမွန္ rule ေတြအျပင္ အဲ့ဒီ့ rule ေတြနဲ႕ပါျပေပးတယ္ဆိုပါေတာ့။

ဒီေတာ့ ဘာကိုေတြ႕ရင္ ဘာလုပ္ရမယ္ဆုိတဲ့ render သို႕ translate လုပ္ေပးတဲ့ engine ေလးေတြ သူတုိ႕မွာရွိၾကတယ္။ အဲ့ဒီ့ engine ေကာင္းရင္ ေကာင္းသေလာက္ သူတုိ႕ရဲ႕ performance ေကာင္းတယ္၊ speed ျမန္တယ္ေပါ့။

ဒါ့အျပင္ အခုအခ်ိန္မွာ web page တစ္ခုကို အသံုးလြယ္ေစဖုိ႕ interactive ျဖစ္ေစဖုိ႕အတြက္ သံုးေနၾကတဲ့ Javascript အတြက္ကိုလည္း support လုပ္တဲ့ engine ကလည္း web browser ေကာင္းမေကာင္းကို affect ျဖစ္တဲ့ အခ်က္တစ္ခုပါပဲ။

ဘယ္ Browser ကို သံုးသင့္သလဲ

ပံုမွန္သံုး user တစ္ေယာက္ အေနနဲ႕ အမ်ားစုသည္ နဂိုကတည္းကပါတဲ့ Internet Explorer(IE) ကိုသံုးၾကပါတယ္။ ဒါေပမယ့္ IE ဟာ အေကာင္းဆံုးလားဆိုေတာ့ မဟုတ္ပါဘူး။ အဆိုးဆံုးထဲမွာေတာင္ ပါတယ္လုိ႕ ေျပာလုိ႕ရပါတယ္။ အခုေနာက္ဆံုး version ျဖစ္တဲ့ vista မွာပါတဲ့ IE7 မွာ အမ်ားၾကီး ေကာင္းေအာင္ လုပ္ထားေပမယ့္လဲ သူ႕ရဲ႕ performance နဲ႕ functionality ဟာ တျခား browser ေတြကို မမွီေသးပါဘူး။

ကၽြန္ေတာ့္အေနနဲ႕ recommend လုပ္ခ်င္တဲ့ Firefox ကေတာ့ performance အရ၊ security အရ၊ speed အရ အမ်ားၾကီး ပိုေကာင္းတ့ဲအျပင္၊ open source ျဖစ္ျပီး developer ေတြ အမ်ားၾကီးရဲ႕ ေက်းဇူးေၾကာင့္ ကုိယ္လုိခ်င္တဲ့ functionality ေတြ အသုံးလုိတဲ့ functionality ေတြကို လြယ္လြယ္ကူကူ free plug-in ေတြ သြင္းျပီး customize လုပ္လုိ႕ ရတဲ့ အတြက္ အခု အခ်ိန္မွာ ဒုတိယ အသံုးအမ်ားဆံုးျဖစ္ေနပါတယ္။ အကယ္၍သာ IE သည္ အသံုးအမ်ားဆံုးျဖစ္တဲ့ Windows နဲ႕ အတူတူပါျပီးသား software တစ္ခု မဟုတ္ခဲ့ရင္ Firefox သည္ လူသံုးအမ်ားဆံုး browser တစ္ခုျဖစ္မွာပါပဲ။
